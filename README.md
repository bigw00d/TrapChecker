TrapChecker
====

Check Trap And Transmit to User.

## Requirement

- Target Board: KL25Z  
- Device: IM920TX   
- IDE: CooCox CoIDE(1.7.8) or MBED

## Usage

- frdm_im920c_gaweway : gateway of house side
- frdm_im920c_rcv : device of user side
- frdm_im920c_snd : device of trap side
